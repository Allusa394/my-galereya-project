# Галерея — план проекта

## Концепция

Онлайн-витрина авторских картин маслом от 3 художников.  
Примерка на своей стене (ArtPlacer), выход на художника через WhatsApp/Telegram.  
Без корзины, без оплаты, без регистрации.

**Воронка:**
```
Instagram / Telegram / VK → Лендинг → Мессенджер художника
```

## Что строим

Один файл `index.html` — все стили, разметка и скрипты внутри.

**Секции страницы:**
1. NAV — фиксированная, лого + 4 якорных ссылки
2. HERO — тёмный фон, крупный заголовок, кнопка в каталог
3. КАТАЛОГ — фильтр по художнику, сетка карточек (6 картин)
4. ПРИМЕРКА — ArtPlacer виджет, «примерь на своей стене»
5. ХУДОЖНИКИ — 3 карточки (фото, имя, биография)
6. ФУТЕР — терракотовый, контакты WhatsApp + Telegram

**Дизайн:** строгий галерейный минимализм. Никаких анимаций — вау-эффект это сами картины.

---

## Технологии

- HTML5 + CSS3 + Vanilla JavaScript
- Google Fonts: Playfair Display (заголовки) + Inter (текст)
- ArtPlacer widget
- Без фреймворков, без npm, без сборки
