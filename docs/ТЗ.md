# Техническое задание — Галерея

**Проект:** Онлайн-витрина авторских картин  
**Дата:** 2026-05-19  
**Версия:** 1.0

---

## 1. Цель проекта

Создать одностраничный лендинг-витрину для продажи оригинальных картин маслом от трёх художников. Сайт работает как витрина: показывает картины, даёт возможность примерить их на своей стене, и отправляет клиента напрямую в мессенджер художника/галереи для оформления покупки.

**Оплата и транзакции на сайте НЕ предусмотрены.**

---

## 2. Целевая аудитория

- Люди после ремонта или переезда — стена пустая, нужна картина
- Хотят оригинал маслом, а не постер
- Бюджет: 5 000 — 80 000 руб. (основной диапазон на витрине)
- Предпочитают не ходить по галереям — хотят удобно выбрать онлайн
- Принимают решение вместе с кем-то (пара, семья) — нужна визуализация

---

## 3. Функциональные требования

### 3.1 Каталог картин
- Карточки картин с крупным фото, названием, именем художника, размером (ШхВ в см) и ценой
- Фильтрация по художнику (Все / Художник 1 / Художник 2 / Художник 3)
- На каждой карточке две кнопки:
  - **«Примерить на стене»** — открывает инструмент примерки
  - **«Написать»** — открывает мессенджер с предзаполненным сообщением

### 3.2 Примерка на стене
- Пользователь нажимает «Примерить на стене» на карточке
- Открывается интерфейс: загрузить фото своей стены
- Картина накладывается на фото в правильном масштабе (по реальным размерам в см)
- Можно двигать и масштабировать положение картины
- Реализация: ArtPlacer виджет (встраиваемый, есть бесплатный план) или canvas JS

### 3.3 Контакт / CTA
- Кнопка «Написать» открывает WhatsApp или Telegram
- Сообщение предзаполнено: «Здравствуйте, интересует картина [Название]»
- Контактная информация в футере

### 3.4 Страницы художников
- Секция с кратким описанием каждого из трёх художников
- Фото художника
- 2–3 предложения: образование, стиль, особенности

### 3.5 Навигация
- Фиксированный nav: логотип + ссылки (Каталог, Художники, Примерка, Контакт)
- Плавный скролл к секциям по клику

## 5. Технологический стек

```
HTML5 + CSS3 + Vanilla JavaScript
— Один файл index.html (всё в одном)
— Никаких фреймворков, никаких npm
— Шрифты: Playfair Display + Inter (Google Fonts CDN)
— Иконки: SVG inline (без библиотек)
— Примерка: ArtPlacer embed widget (JS snippet)
```

---

## 6. Структура страницы (секции)

```
[NAV]          — Лого + ссылки (fixed top)
[HERO]         — Заголовок, подзаголовок, CTA-кнопка
[CATALOG]      — Фильтры + сетка карточек картин
[WALL TRYOUT]  — Промо-блок функции примерки
[ARTISTS]      — 3 профиля художников
[FOOTER/CTA]   — Контакты, WhatsApp/Telegram, копирайт
```

## 7. Дизайн-система

### Цвета
```

Фон инверсный:       #1A1A1A  (почти чёрный)
Текст основной:      #1A1A1A

Текст инверсный:     #FFFFFF
Граница:             #C5BEB6
Акцент (терракота):  #D4916E
```

### Типографика
```
Заголовки:  Playfair Display, Regular 400 / Bold 700
Текст:      Inter, Regular 400 / SemiBold 600
Размеры:    72px / 48px / 32px / 22px / 18px / 14px / 12px
```

### Отступы
```
Padding секций:   80px (desktop) / 24px (mobile)
Gap карточек:     24px
Радиус кнопок:    4px
Радиус карточек:  4px
```
