# Структура файлов проекта

```
my-galereya-project/
│
├── index.html              ← Единственный HTML-файл (весь фронтенд)
│
├── images/
│   ├── paintings/
│   │   ├── painting-01.jpg
│   │   ├── painting-02.jpg
│   │   └── ...
│   └── artists/
│       ├── artist-1.jpg
│       ├── artist-2.jpg
│       └── artist-3.jpg
│
├── docs/                   ← Технические документы
│   ├── ТЗ.md
│   ├── структура-файлов.md
│   └── контент.md
│
├── project.md              ← Описание проекта
├── research.md             ← Исследование + критика
├── PLAN.md                 ← План работ
└── CLAUDE.md               ← Правила работы с ИИ
```

## Принципы

- **Один HTML-файл** — CSS и JS встроены внутрь `<style>` и `<script>`
- **Нет сборщиков** — никакого Webpack, Vite, npm
- **Нет фреймворков** — только ванильный JS
- **Нет внешних зависимостей** — только Google Fonts CDN и ArtPlacer виджет
