# Как обновлять контент галереи с телефона

## Два варианта

| Вариант | Как | Когда использовать |
|---|---|---|
| **А — через Claude** | Сказать Claude что изменить в `data.json` | Редкие правки, просто сказать |
| **Б — Google Sheets** | Открыть таблицу → изменить → сохранится само | Регулярные обновления самостоятельно |

Переключение в `index.html`: `const DATA_SOURCE = 'sheets'` или `'json'`

---

## Вариант Б — Google Sheets (основной)

### Ссылка на таблицу
Таблица опубликована как CSV, сайт читает её при загрузке:
```
Base URL: https://docs.google.com/spreadsheets/d/e/2PACX-1vTU3Bnk-aod6ZcUcHhWZWxGsILWRXi_Hci93uZOU01x_Ll_0CZ0avboStXDGaQSWyGTrBtZ11If5hts/pub?output=csv
Paintings GID: 1804990058
Artists GID:   1863760401
```

### Как изменить данные
1. Открыть Google Sheets на телефоне
2. Найти нужную строку (картину или художника)
3. Изменить ячейку → нажать Enter
4. Таблица сохраняется **автоматически** (вверху: "Все изменения сохранены")
5. Открыть сайт → обновить страницу → изменение появится

### Структура листа Paintings
| Колонка | Что писать | Пример |
|---|---|---|
| id | уникальный код | p01, p02... |
| title | название картины | Осенний пейзаж |
| artist | id художника | artist-1 |
| artistLabel | имя для показа | Имя Фамилия |
| technique | техника | Масло, холст |
| width | ширина в см | 80 |
| height | высота в см | 60 |
| price | цена в рублях (0 = "Цена по запросу") | 25000 |
| imageUrl | прямая ссылка на фото | https://i.imgur.com/... |
| wa | номер WhatsApp (без +) | 79991234567 |
| tg | ник Telegram (без @) | username |

### Структура листа Artists
| Колонка | Что писать |
|---|---|
| id | artist-1, artist-2, artist-3 |
| name | Имя Фамилия |
| photoUrl | ссылка на фото художника |
| bio | 2–3 предложения о художнике |

## Как загрузить фото картины

### Вариант 1 — Imgur (без регистрации, быстро)
1. Зайти на `imgur.com` с телефона
2. Нажать Upload → выбрать фото
3. Скопировать ссылку (вида `https://i.imgur.com/AbCdEfG.jpg`)
4. Вставить в столбец `imageUrl` в Google Sheets

### Вариант 2 — Cloudinary (надёжнее, нужна регистрация)
1. Зарегистрироваться на `cloudinary.com` (бесплатно, 25 ГБ)
2. Загрузить фото → скопировать URL
3. Вставить в столбец `imageUrl` в Google Sheets

## Локальный сервер (для разработки)
Открывать `index.html` напрямую нельзя — fetch не работает через file://.

Запустить сервер:
```
npx serve . --listen 3000
```
Затем открыть: `http://localhost:3000`
