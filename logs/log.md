## 2026-05-19 — fetch не работает при открытии index.html напрямую
- **Ошибка:** Страница пустая, в консоли `fetch` blocked, CORS error
- **Причина:** Браузер блокирует fetch() по протоколу `file://`
- **Решение:** Запускать локальный сервер: `npx serve . --listen 3000`, открывать `http://localhost:3000`
- **Статус:** ✅ Исправлено (workaround — всегда использовать сервер)

## 2026-05-19 — Импорт CSV создал отдельную таблицу вместо нового листа
- **Ошибка:** При импорте Paintings.csv Google Sheets создал новый файл "Paintings" вместо добавления листа в существующую таблицу
- **Причина:** В диалоге импорта была выбрана опция "Создать таблицу" вместо "Заменить текущий лист"
- **Решение:** В диалоге импорта кликнуть на выпадающий список и выбрать нужную опцию
- **Статус:** ✅ Принято как есть (данные корректны)
