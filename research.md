# Исследование ниши — Шаг 2 по алгоритму

> Алгоритм: ИДЕЯ → ИССЛЕДОВАНИЕ → **КРИТИКА** → ПЛАН → РЕАЛИЗАЦИЯ

**Идея проекта:**
Я создаю онлайн-магазин картин от трёх художников с примеркой на стене для людей, которые хотят повесить дома настоящую картину (масло, оригинал, 5 000–250 000 руб.), чтобы решить проблему выбора — можно увидеть, как картина смотрится у тебя, и заказать в один шаг.

## 1. ИССЛЕДОВАНИЕ — 7 похожих продуктов

### 1. Smart Gallery (gallerysmart.ru)
**Что делает:** Онлайн-магазин оригинальных картин, «музейный уровень», доставка по Москве.
**Хорошо:**
- Сильный оффер «бесплатная примерка и доставка по Москве»
- Акцент на качество — слова «музейный уровень» снимают сомнения
- Каталог отфильтрован по стилю, размеру, цене

**Плохо:**
- Примерка через консультанта, не самостоятельно (нет онлайн-инструмента)
- Упор на Москву — жёсткое гео-ограничение
- Много художников → размытый бренд, нет личной истории

**Взять:** Формулировку «бесплатная примерка» как доверие-триггер; фильтрацию по размеру и стилю.

### 2. ArtWorld.ru
**Что делает:** Онлайн-галерея с примеркой картин в интерьере, доставка по России и миру.
**Хорошо:**
- Онлайн-примерка работ в интерьере (готовят визуализацию)
- Доставка по всей России — нет Москва-центричности
- Широкий ассортимент

**Плохо:**
- Примерка не self-service: нужно ждать, пока сделают вручную
- Сотни художников → покупатель теряется
- Дизайн устаревший, нет ощущения доверия

**Взять:** Акцент на доставке по всей России; идею «пришли фото стены — получишь визуализацию» как запасной сценарий.

### 3. itisart.ru
**Что делает:** Магазин 100% оригинальных картин, подписанных художниками лично.
**Хорошо:**
- Сильный УТП: «каждая картина уникальна, без повторов и копий»
- Личная подпись художника — эмоциональная ценность, сертификат подлинности
- Убирает страх «а вдруг это копия»

**Плохо:**
- Нет примерки на стене совсем
- Много художников → нет истории, нет лица бренда
- Цены не очевидны на первый взгляд

**Взять:** Идею сертификата подлинности + личная подпись художника как элемент доверия на карточке товара.

### 4. DasArt.ru
**Что делает:** Оригинальные картины + репродукции, 200+ вариантов рам, примерка рамы онлайн.
**Хорошо:**
- Инструмент примерки рам — конкретный, полезный

**Плохо:**

- Нет примерки самой картины на стене клиента
- Нет историй о художниках

**Взять:** Идею выбора рамы к картине как дополнительный сервис (upsell).

**Хорошо:**

**Плохо:**

- Нет самостоятельного онлайн-инструмента
- Слабый дизайн сайта

**Взять:** Формулировку «под ваш интерьер» — это и есть главная боль покупателя.

### 6. Saatchi Art (saatchiart.com) — мировой лидер
**Что делает:** Маркетплейс 100 000+ художников, оригиналы и принты, доставка по миру.
**Хорошо:**
- Инструмент «View In a Room» — self-service примерка в готовых интерьерах
- Доверие: большой бренд, огромный каталог
- Фильтры по стилю, цвету, размеру, цене
- Чёткая бизнес-модель: 40% комиссия с продаж художника

**Плохо:**
- Нет примерки на СВОЕЙ стене — только в шаблонных комнатах
- 1,4 млн картин → невозможно выбирать, покупатель тонет
- Английский язык, нет рублёвых цен, нет понимания российского рынка
- Художники анонимны — нет личной связи

**Взять:** Кнопку «Посмотреть на стене» прямо в каталоге; фильтры по цвету (под интерьер); чёткие размеры с линейкой масштаба.

### 7. ArtPlacer (artplacer.com) — инструмент для галерей
**Что делает:** SaaS-инструмент примерки произведений искусства в 2800+ комнатах или на фото клиента.
**Хорошо:**
- Самый мощный инструмент примерки: масштаб, перспектива, цвет стены
- Можно загрузить СВОЁ фото интерьера — самая реалистичная примерка
- Встраивается на любой сайт как виджет
- Бесплатный план для небольших галерей

**Плохо:**
- Это B2B инструмент (для галерей), не для покупателей
- Платный для серьёзного использования

**Взять:** ArtPlacer можно встроить на сайт галереи — это и есть та самая «примерка на стене» из идеи! Бесплатный план покрывает старт.

## 2. КРИТИКА ИДЕИ — честная оценка

### 3 слабых места:

**1. Примерка — ожидания vs реальность**
Большинство конкурентов обещают «примерку» и делают её через консультанта или в шаблонных комнатах. Self-service примерка на своей стене — это технически сложно и непривычно для российского покупателя. Если реализовать плохо — разочарование убьёт конверсию.

**2. 3 художника — маленький каталог**
При 5 000–250 000 руб. диапазоне и всего 3 художниках ассортимент будет ограниченным. Если у конкретного человека нет «своей» картины в каталоге — он уйдёт. Маленький каталог = маленький трафик из поиска.

**3. Нет доверия к новому бренду**
Купить картину за 50 000–250 000 руб. у незнакомого сайта — высокий психологический барьер. Конкуренты берут доверие через годы работы, отзывы, физические галереи. У нового сайта этого нет.

### Что не учтено:
- **Логистика** — картины на холсте хрупкие, дорогая упаковка и доставка, возвраты сложны
- **Кто принимает решение о покупке?** Часто пара, не один человек — как убедить двоих?
- **Цикл сделки** — картина за 100к не покупается с первого визита, нужны касания (email, мессенджер)

### Кому это точно НЕ нужно:
- Тем, кто хочет постер или репродукцию (не оригинал)
- Тем, кто покупает «подарок» без понимания вкусов получателя
- Тем, у кого нет стены — съёмное жильё, временный ремонт

## ШАГ 3 — КРИТИКА

### Что может не сработать
1. **Примерка — главная фича, которую легко сломать.** Если накладывается криво или медленно — пользователь не доверяет и уходит. Работает только если работает идеально.
2. **3 художника не закрывают все вкусы.** Два академиста + один интерьерный — перекос в стиле. Современный минимализм, «цветы в спальню» — не найдут ничего.
3. **250 000 руб. онлайн без физического контакта.** Верхний сегмент покупают после живого знакомства. Онлайн продаётся хорошо до 30–50k, дальше нужен другой процесс.
4. **WhatsApp/Telegram как единственный CTA — это барьер.** Писать незнакомому человеку не хочется. Нужна простая форма «Оставить заявку».
5. **Нет повторных покупок.** Картину покупают раз в несколько лет. Бизнес без повторных продаж живёт только на трафике — откуда он будет?

### Чего не хватает
- Социальное доказательство: отзывы, фото покупателей с картиной в реальном интерьере
- История художников: без личной истории картина = товар, а не произведение
- Механизм доверия при высокой цене: возврат? Сертификат подлинности?
- Путь для «смотрю, но не готов купить»: подписка, избранное, email-захват
- SEO-структура: одностраничный лендинг не ранжируется по запросам «купить картину маслом»

### Кому точно НЕ нужно
- Тем, кто хочет постер до 2 000 руб.
- Тем, кто покупает «в подарок» без знания вкусов получателя
- Тем, кто в активном ремонте — «потом»
- Корпоративным клиентам — у них тендерный процесс

### Улучшения
1. **Сузить диапазон на лендинге:** 5 000–80 000 руб. Дорогие работы — по запросу «написать художнику»
2. **Первый шаг без мессенджера:** кнопка «Зарезервировать» с формой на 2 поля
3. **Блок «Истории покупок»:** «Анна, Петербург — взяла пейзаж в гостиную, вот как смотрится» + фото
4. **SEO-страницы:** `/kartiny-dlya-gostinoj`, `/portret-na-zakaz` — трафик без бюджета
5. **Email/Telegram-подписка:** захват тех, кто смотрит но не готов

### Обновлённая формулировка идеи после критики
> Я создаю онлайн-магазин авторских картин маслом (3 художника, оригиналы, 5 000–80 000 руб.) для людей после ремонта или переезда в Петербурге и Москве, которые хотят настоящую картину, а не постер — с функцией «посмотри как смотрится у тебя на стене» и простым заказом без регистрации, чтобы убрать главный страх: «куплю — не подойдёт».
