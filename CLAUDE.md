# Правила работы с проектом галереи

## ГЛАВНОЕ ПРАВИЛО — Документация в проекте
Вся документация хранится в файлах ЭТОГО проекта. После каждого важного решения или реализации — сразу обновить нужный файл.

**Карта документации:**
- `PLAN.md` — план и текущий статус всех этапов (обновлять галочки)
- `docs/ТЗ.md` — техническое задание
- `docs/КАК-ОБНОВЛЯТЬ.md` — инструкция по обновлению контента с телефона
- `docs/структура-файлов.md` — структура проекта
- `data.json` — данные картин и художников (Вариант А)
- `sheets-import/` — CSV шаблоны для Google Sheets
- `logs/log.md` — лог всех ошибок и их решений
- `docs/DEPLOYMENT.md` — хронология событий по проекту
