# Галерея — живопись для вашего дома

## Что это
Онлайн-магазин картин от трёх художников с возможностью примерить картину на своей стене и заказать работу под свой интерьер.

## Суть в одном предложении
Я создаю онлайн-магазин картин от трёх художников с примеркой на стене для людей, которые хотят повесить дома настоящую картину, чтобы решить проблему выбора — можно увидеть, как картина смотрится у тебя, и заказать в один шаг.

## Художники
- **Художник 1** — известный автор, интерьерная живопись. Не академист. Доступный ценовой сегмент.
- **Художник 2** — выпускник Академии художеств им. Репина. Академическая школа. Средний/высокий сегмент.
- **Художник 3** — выпускник Академии художеств им. Репина. Академическая школа. Средний/высокий сегмент.

## Товар
- Все картины — масло на холсте, оригиналы
- Ценовой диапазон: 5 000 — 250 000 руб.
- Есть возможность заказать картину под свои пожелания (размер, цвета, сюжет)

## Для кого
- Люди, которые хотят настоящую картину домой, а не постер
- После ремонта или переезда — стена пустая, нужна картина
- Те, кто хочет оригинал, но не хочет ходить по галереям
- Покупатели с бюджетом от 5 000 до 250 000 руб.

## Что есть на сайте
- **Каталог картин** — крупные фото, размер, цена, фильтр по художнику
- **Примерка на стене** — загружаешь фото своей стены, видишь как картина смотрится
- **Страницы художников** — коротко о каждом авторе и его работы
- **Заказ под интерьер** — можно заказать картину по своим пожеланиям
- **Контакт** — WhatsApp / Telegram для вопросов и заказов

## Чего НЕ будет
- Не маркетплейс — только 3 конкретных художника
- Не принты и постеры — только оригиналы маслом на холсте
- Не AI-сервис — примерка на стене, но без лишних buzzwords
- Не сложная регистрация — покупка без аккаунтов
- Не галерея с пафосом — простой и понятный магазин

## Технологии
- Одностраничный лендинг: HTML + CSS + JS (всё в одном файле)
- Шрифты: Playfair Display (заголовки) + Inter (текст)
- Стиль: minimal gallery, светлый, много воздуха, крупные картины
- Цвета: белый #FFFFFF, молочный #F7F5F2, серый #EAEAEA, текст #111111, акцент терракота #C47A5A
- SVG для иконок, без внешних JS-библиотек
